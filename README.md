# Sign Language Gesture Recognition

A real-time sign language gesture recognition system that uses Computer Vision and Machine Learning to detect hand gestures from a webcam and convert them into text.

## Technologies
- Python
- OpenCV
- MediaPipe
- NumPy
- Scikit-learn

## Features
- Real-time webcam input
- Hand landmark detection
- Gesture classification
- Gesture-to-text conversion
