# Sign Language Gesture Recognition

A real-time Sign Language Gesture Recognition System developed using Python, OpenCV, MediaPipe, and Machine Learning.

## Features

* Real-time webcam-based gesture detection
* Hand landmark extraction using MediaPipe
* Custom gesture dataset creation
* Machine Learning model training using Random Forest
* Real-time gesture prediction

## Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy
* Pandas
* Scikit-learn

## Project Workflow

Webcam Input → Hand Detection → Landmark Extraction → Model Prediction → Gesture Display

## Supported Gestures

* YES 👍
* NO 👎
* STOP ✋
* OK 👌
* HELLO 👋

## Future Improvements

* Increase dataset size for higher accuracy
* Add more sign language gestures
* Implement text-to-speech conversion
* Improve model performance with deep learning
